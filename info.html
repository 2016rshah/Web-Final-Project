Basics

  • Click anywhere on the canvas to create a dot
  • Select a dot by clicking on it, or dragging a rectangle over it
  • Hold the control key to select multiple dots at a time
  • Move selected dots by dragging the mouse or using the arrow keys
  • The Clear button clears the canvas
  • The Undo button undoes the last move, the Replay button replays all actions

Edges

  • Given a set of selected vertices S, E connects every vertex in S to every vertex in the complement of S
  • C clears all edges with one dot in S and one dot in the complement of S
  • T toggles all such vertices
  • Edges can be selected the same way as dots
  • Drag a point on an edge to curve it

Properties

  • Add a new property option using the Add Property input box and button
  • Set a property for a set of selected dots/edges using the Set Property Value drop-down menus, input box, and button
  • Mouse over a dot/edge to show property values
  • Captions are a default property; when it is set, it is shown next to the item it is labeling without needing to mouse over

Sequence Mode

  • Click the sequence button to enter sequence mode
  • In sequence mode, every new dot created will automatically create an edge between the new and previous vertex
  • Mode is automatically terminated once the original dot is reached
  • Mode is terminated by clicking the button or pressing q
  • Escape exits cancelling changes

Spoke Mode

  • Click the edge button to enter spoke mode
  • In spoke mode, a vertex can be created by clicking on an empty spot
  • First, select a vertex(ices) to be the indicated vertex(ices)
  • Then, the vertex(ices) that the mouse selects for will toggle edges between the originally indicated vertices, and the latest selected vertices
  • Mode is terminated by clicking the button, or pressing q
  • Escape exits cancelling changes

Algorithms

  • Click the Dijkstra button to find the shortest path between two vertices, taking into account the weights of the edges. If a weight is not set, the edge size is used as default
  • The shortest path will be highlighted in red

Saving/Retrieving Canvases

  • Press save to save the current canvas
  • Press load to load your previously saved canvas
